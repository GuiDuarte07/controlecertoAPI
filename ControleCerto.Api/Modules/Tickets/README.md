# Módulo Tickets (Chamados/Ajuda)

- `Controllers/`: endpoints REST para usuário (`api/tickets`) e admin (`api/admin/tickets`).
- `DTOs/`: contratos de request/response (inclui upload via `multipart/form-data`).
- `Services/`: regras de negócio do chamado, anexos (S3) e disparo de notificações/e-mails.
